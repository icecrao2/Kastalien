# 06-4 뷰 바인딩
뷰 바인딩이란 XML 파일에 선언한 뷰 객체를 코드에서 쉽게 이용하는 방법이다.

- 기존에는 findByViewId 함수를 통해 액티비티에 등록된 뷰 객체를 불러올 수 있었지만 이 작업이 너무 귀찮았었고 그래서 나온것이 뷰 바인딩이다.
- 우선 뷰 바인딩을 사용하기 위해서는 gradle 설정이 되어야 한다.
    - gradle에 뷰 바인딩 설정을 하면 layout XML파일에 등록된 뷰 객체를 포함하는 클래스가 자동으로 만들어진다.
    ```gradle
        android {
            vieweBinding.isEnabled = true
        }
    ```
    - 이렇게 만들어진 뷰 객체 클래스의 네임은 XML파일명을 따른다.
        - activity_main.xml -> ActivityMainBinding
        - item_main.xml -> ItemMainBinding
    - 자동으로 만들어진 클래스의 inflate 함수를 호출하면 바인딩 객체를 얻을 수 있다.
    - 만약에 특정 xml에서 뷰바인딩을 사용하지 않겠다면 해당 xml의 루트 태그에 tools:viewbindingIgnore="true" 를 추가하면 된다.
