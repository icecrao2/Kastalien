# 06-3 기본적인 뷰 살펴보기
## 텍스트 뷰
문자열을 화면에 출력하는 뷰
- android:text 속성
    - 출력할 문자열을 지정한다.
    - android:text="hello" 처럼 문자열을 대입 가능하다.
    - android:text="@string/hello" 처럼 문자열 리소스를 지정해도 된다.
- android:textColor 속성
    - 문자열의 색상을 지정한다.
    - android:textColor="FF0000" 처럼 16진수 RGB 형식을 사용한다.
    - +alpha(!질문!) android:text 속성처럼 리소스를 지정해도 되나?
- android:textSize 속성
    - 문자열의 크기를 지정한다.
    - android:textSize="20sp" 처럼 숫자를 사용하며 단위는 생략 불가능하다.
    - +alpha(!질문!) 마찬가지로 android:text 속성처럼 리소스를 지정해도 되나?
- android:textStyle 속성
    - 문자열의 스타일을 지정한다.
    - android:textStyle="bold" 처럼 사용한다.
        - 값은 bold, italic, normal 중에서 선택한다.
- android:autoLink 속성
    - TextView에 출력할 문자열을 분석해 특정 형태의 문자열에 자동 링크를 추가해준다.
    - 예) android:autoLink="web"으로 지정하고 android:text=웹주소 를 하면 해당 문자열을 링크 모양으로 표시하고 사용자가 클릭하면 웹 브라우저를 띄울 수 있게 한다.
    - 속성
        - web
        - email
        - phone
        - ...
- android:maxLines 속성
    - TextView는 문자열을 출력할 때 자동으로 긴 문자열은 줄바꿈을 한다.
    - 때로는 문자열이 특정 줄까지만 나오도록 해야 할 때가 있고 이 때 사용하는것이 android:maxLines 속성이다.
- android:ellipsize 속성
    - 문자열이 크기를 초과했을때 ... 같이 문자열이 더 있음을 나타낼때 사용한다.
    - 속성
        - start
            - 문자열 앞부분에 ...이 추가된다
            - singleline일 경우에만 사용 가능
        - middle
            - 문자열 중간부분에 ...이 추가된다.
            - singleline일 경우에만 사용 가능
        - end
            - 문자열 마지막부분에 ...이 추가된다.

## 이미지뷰
이미지를 출력하는 View이다.
- android:src 속성
    - ImageView에 출력할 이미지를 설정한다.
        - android:src="@drawable/image3"
- android:maxWidth / android:maxHeight / android:adjustViewBounds
    - ImageView가 출력하는 이미지의 최대 크기를 지정한다.
    - maxHeight, maxWidth 속성은 adjustViewBounds 속성과 함께 사용되어야 한다.
    - adjustViewBounds 속성을 true로 설정하면 이미지의 가로 세로 길이와 비례해 뷰의 크기를 맞춘다.

## 버튼, 체크박스, 라디오버튼
- button
    - 버튼은 사용자 이벤트를 처리한다.
- checkbox
    - 다중 선택을 가능하게 한다.
- radiobutton
    - 단일 선택을 제공한다.
    - radiogroup과 함께 사용된다.

## 에디트 텍스트
사용자가 글을 입력할 수 있는 뷰이다.
- android:lines, android:maxLines 속성
    - android:lines="3"을 하면 처음부터 3줄 크기로 에디트 텍스트가 화면에 나오며 그 이상 늘어나지 않는다.
    - android:maxLines="3"을 하면 처음에는 한 줄 입력 크기로 출력되지만, 엔터를 누르면 3줄까지 늘어나고 그 이상 늘어나지 않는다.
- android: inputType 속성
    - 에디트 텍스트에 글을 입려갈 때 올라오는 키보드를 지정하는 속성이다.
