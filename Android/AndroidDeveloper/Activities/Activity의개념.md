# Activity의 개념

### 개념

- 검색을 통한 Activity의 개념은 Android의 클래스이며 앱의 진입점이 되는 Window 라고 표현된다.
    - Activity는 앱의 화면을 표현하는 개념이 아닌 프로그램을 표현하는 개념으로 해석된다.
- 다른 프레임워크들과는 달리 Activity가 앱의 진입점이 됨으로 다른 프레임워크에서 사용되는 main 같은 단일 진입점 개념이 사라진다.
